# SmartERP AI

An AI-powered Enterprise Resource Planning platform for SMEs in Cameroon.

## Getting Started

The application is built with Next.js and utilizes Genkit for AI functionalities.

### Project Structure

- `src/app/`: Root routes and core modules (Dashboard, Inventory, Sales, Tasks, AI Assistant).
- `src/ai/`: Genkit flow definitions and AI logic.
- `src/components/`: Reusable UI components powered by ShadCN.
- `src/lib/`: Types, mock data, and utility functions.

## Features

- **Multi-Tenant SaaS**: Isolated business data per tenant.
- **Inventory Management**: Real-time stock tracking and low-stock alerts.
- **Sales Tracking**: Transaction logs and financial analytics.
- **Task Management**: Operational workload monitoring and overdue detection.
- **AI Assistant**: Context-aware business intelligence (Read-only).
